# Note App

Note App using Node.JS

## Installation

Use the node package manager npm

```bash
npm init
npm install express
npm install babel-polyfill
npm install body-parser
npm install memorystorage
npm install sequential-ids
npm install nodemon
npm install cors

## Usage

```node
postman for testing
```
